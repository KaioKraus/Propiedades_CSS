Display

Display Block:

O display block é uma propriedade que faz um elemento HTML ocupar 100% da largura do contêiner pai, forçando a quebra de linha, sendo possível de usar as estilizações: width, height, margin e padding

Display inline:

É usado para que elementos fiquem lado a lado na mesma linha, ocupando apenas a largura necessária para o seu conteúdo, sendo ignorado as estilizações de width e height.

Display inline-block:

É usado quando quer manter os elementos filhos lado a lado, assim como o display inline, porém é possível de usar as estilizações de width, height, padding e margin. Assim alterando o tamanho do contêiner pai e não dos elementos filhos.

Display flex

O display flex cria um container flexível, alinhando os elementos filhos em colunas ou linhas. Atualmente é a forma mais eficiente de fazer um layout responsivo.

Display inline-flex:

Ele é uma mistura entre o display inline e o flex, assim transformando o elemento pai em um contêiner flex porém ocupando apenas o espaço que necessário para que ele exista.

Display grid:

O display grid faz o elemento pai ter uma forma de tabela, tendo colunas e linhas, assim sendo possível ter um maior controle e uma precisão maior para os elementos filhos, podendo ser criados estruturas mais complexas de uma maneira mais fácil.

Display inline-grid:

O display inline grid é usado para transformar um elemento em HTML em um contêiner de grade (grid container) que se comporta como um elemento inline, ou seja, o elemento passa a não ocupar toda a largura da linha.

Display flow root

O display flow root é utilizado no CSS para criar um novo contexto de formatação de bloco (BFC), fazendo com que um contêiner envolva totalmente seus filhos flutuantes, ele também impede que elementos flutuantes passem o limite inferior de contêiner e assim evitando o colapso de margens com elementos externos

Display table:

O display table é usado no CSS e serve para que elementos no HTML como uma div ou uma section vire uma tabela, as organizando como linhas e células.

Display block flow:

O display block flow é usado em casos em que é necessário que um elemento HTML precise ocupar 100% da largura disponível, assim gerando quebra de linhas antes e depois dele.

Display Block flex:

O display block flex é usado para criar um layout que seja responsivo e para alinhamentos complexos de forma simples e rápida, ele faz isso transformando um elemento em um container flexível

Display Block grid:

O display block grid serve para criar um layout bidimensional, organizando os elementos filhos de forma simultânea em linhas e colunas, ele transforma o container em uma grade, assim possibilitando um alinhamento preciso.

Display flow-root:

O display block flow-root serve para criar um contexto de formatação de bloco no css, ele serve basicamente como um “display:block”, mas isola o elemento, assim impedindo que flutuadores afetem o layout.

Display inline flow:

O display inline flow é usado para renderizar elementos na mesma linha, colocando um do lado do outro, e ele faz isso sem forçar uma quebra de linha antes ou depois, ele ocupa somente a largura necessária para o conteúdo.

Display inline flex:

O display inline flex é usado para criar um contêiner que se comporta como um elemento em linha, ou seja, ele ocupa somente a largura necessária , possibilitando elementos ao lado, ele é ideal para alinhar itens lado a lado sem quebrar linhas.

Display Inline grid:

O display inline grid é utilizado no CSS para criar um contêiner de grade que se comporta como um elemento inline no fluxo do documento, ao invés de um elemento em bloco.

Display inline flow-root:

O display inline flow-root é usado para criar um contêiner que se comporta como um elemento inline e que também estabelece um novo contexto de formatação do bloco
